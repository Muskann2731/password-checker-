# Password Strength Checker 🔐

This is a simple web-based password strength checker built using HTML, CSS, and JavaScript.

### Features:
- Detects password strength: Weak, Medium, or Strong
- Checks for:
  - Password length
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Clean and responsive UI
